Stock Rom installation

To put the device into Fastboot mode, while the device is off, press and hold the Volume Down (-) and Power Button. When the Mi Screen appears, release the Power button. When the Fastboot text appears, release the Volume Down (-) Button.

Unzip the stock ROM file. Run the Xiaomi Flash Tool program. Select the ROM File, select the clean all box and press the flash button.

Stock rom installation completed.

Move the files to "C:\Program Files (x86)\Minimal ADB and Fastboot"
